## dynamic keyword (veri tipi)
- dynamic keywordu runtime çalışıyor.
- dynamic (Dinamik) veri tipi, her tür veriyi depolayabilen bir özel tür daha vardır, ancak nesneden çok esnekliği performans maliyetine bağlıdır.
